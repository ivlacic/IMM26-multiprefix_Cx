Dataset and R code for the study "Linearization in cumulative intensifying prefixation: An information-theoretic perspective", to be presented at the 22nd International Morphology Meeting. 
The study investigates whether two information-theoretic measures -- contextual entropy and paradigmatic surprisal -- predict linearization preferences (i.e., affix ordering) in Italian multi-prefix intensifying constructions.
Authors: Ivan Lacić and Martina Verdelli.
